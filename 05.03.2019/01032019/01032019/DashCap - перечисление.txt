DashCap - перечисление

Задает тип графической формы, используемой для обоих концов каждого штриха в пунктирной линии.
Пространство имен:  System.Drawing.Drawing2D

Flat	Задает квадратное завершение для обоих концов каждого штриха.
Round	Задает круглое завершение для обоих концов каждого штриха.
Triangle	Задает треугольное завершение для обоих концов каждого штриха.
