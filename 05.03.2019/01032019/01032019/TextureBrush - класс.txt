Конструкторы
  	Имя	Описание
	TextureBrush	Перегружен. Инициализирует новый объект TextureBrush, использующий указанное изображение.

////////////////////////////

Методы
  	Имя	Описание
	Clone	Создает точную копию объекта TextureBrush. (Переопределяет Brush.Clone.)
	CreateObjRef	Создает объект, который содержит всю необходимую информацию для создания прокси-сервера, используемого для взаимодействия с удаленным объектом. (Унаследовано от MarshalByRefObject.)
	Dispose	Перегружен. 
	Equals	Определяет, равен ли заданный объект Object текущему объекту Object. (Унаследовано от Object.)
	Finalize	Перегружен. 
	GetHashCode	Играет роль хэш-функции для определенного типа. (Унаследовано от Object.)
	GetLifetimeService	Извлекает объект обслуживания во время существования, который управляет политикой времени существования данного экземпляра. (Унаследовано от MarshalByRefObject.)
	GetType	Возвращает объект Type для текущего экземпляра. (Унаследовано от Object.)
	InitializeLifetimeService	Возвращает объект обслуживания во время существования для управления политикой времени существования данного экземпляра. (Унаследовано от MarshalByRefObject.)
	MemberwiseClone	Перегружен. 
	MultiplyTransform	Перегружен. Умножает объект Matrix, представляющий локальное геометрическое преобразование объекта TextureBrush, на указанный объект Matrix в заданном порядке.
	ResetTransform	Возвращает свойству Transform объекта TextureBrush единичное значение.
	RotateTransform	Перегружен. Поворачивает локальное геометрическое преобразование объекта TextureBrush на заданную величину. Этот метод добавляет поворот перед преобразованием.
	ScaleTransform	Перегружен. Изменяет масштаб локального геометрического преобразования объекта TextureBrush на заданные значения. Этот метод вставляет изменение масштаба перед преобразованием.
	SetNativeBrush	В производном классе определяет ссылку на объект-кисть GDI+. (Унаследовано от Brush.)
	ToString	Возвращает объект String, который представляет текущий объект Object. (Унаследовано от Object.)
	TranslateTransform	Перегружен. Выполняет перенос локального геометрического преобразования объекта TextureBrush на заданные значения в указанном порядке.

/////////////////////////////

Свойства
  	Имя	Описание
	Image	Получает объект Image, связанный с объектом TextureBrush.
	Transform	Получает или задает копию объекта Matrix, определяющую локальное геометрическое преобразование для изображения, связанного с объектом TextureBrush.
	WrapMode	Получает или задает перечисление WrapMode, задающее режим переноса для объекта TextureBrush.
