Конструкторы

/////////////////////


  	Имя	Описание
	ResourceManager	Перегружен. Инициализирует новый экземпляр класса ResourceManager.


Методы

/////////////////////////////


  	Имя	Описание
	CreateFileBasedResourceManager	Возвращает объект ResourceManager, который просматривает конкретный каталог для ресурсов, вместо просмотра манифеста сборки.
	Equals	Определяет, равен ли заданный объект Object текущему объекту Object. (Унаследовано от Object.)
	Finalize	Позволяет объекту Object попытаться освободить ресурсы и выполнить другие операции очистки, перед тем как объект Object будет утилизирован в процессе сборки мусора. (Унаследовано от Object.)
	GetHashCode	Играет роль хэш-функции для определенного типа. (Унаследовано от Object.)
	GetNeutralResourcesLanguage	Возвращает объект CultureInfo для ресурсов нейтральной главной сборки, считывая значение объекта NeutralResourcesLanguageAttribute на заданном объекте Assembly.
	GetObject	Перегружен. Возвращает значение указанного ресурса Object для текущего языка и региональных параметров.
	GetResourceFileName	Создает имя файла ресурсов для заданного объекта CultureInfo.
	GetResourceSet	Возвращает объект ResourceSet для определенного языка и региональных параметров.
	GetSatelliteContractVersion	Возвращает объект Version, указанный посредством SatelliteContractVersionAttribute, в заданную сборку.
	GetStream	Перегружен. Возвращает объект UnmanagedMemoryStream из MemoryStream.
	GetString	Перегружен. Получает указанный ресурс String для заданного языка и региональных параметров или неявное значение текущего языка и региональных параметров пользовательского интерфейса.
	GetType	Возвращает объект Type для текущего экземпляра. (Унаследовано от Object.)
	InternalGetResourceSet	Предоставляет реализацию для обнаружения ResourceSet.
	MemberwiseClone	Создает неполную копию текущего объекта Object. (Унаследовано от Object.)
	ReleaseAllResources	Сообщает ResourceManager о необходимости вызова Close для всех объектов ResourceSet и освобождения всех ресурсов.
	ToString	Возвращает объект String, который представляет текущий объект Object. (Унаследовано от Object.)


////////////////////////////////


Поля

///////////////////////////////


  	Имя	Описание
	BaseNameField	Указывает имя корневой папки файлов ресурсов, в которой класс ResourceManager ищет ресурсы.
	HeaderVersionNumber	Константное значение readonly, указывающее версию заголовков файлов ресурсов, которую текущая реализация объекта ResourceManager может интерпретировать и создавать.
	MagicNumber	Содержит номер, используемый для идентификации файлов ресурсов.
	MainAssembly	Указывает главный класс Assembly, содержащий ресурсы.
	ResourceSets	Содержит объект Hashtable, возвращающий отображение языков и региональных параметров в объекты ResourceSet.


Свойства

////////////////////////////////


  	Имя	Описание
	BaseName	Возвращает имя корневой папки файлов ресурсов, в которой класс ResourceManager ищет ресурсы.
	FallbackLocation	Возвращает или задает местоположение, из которого извлекаются нейтральные резервные ресурсы.
	IgnoreCase	Возвращает или задает логическое значение, которое указывает, позволяет ли текущий экземпляр ResourceManager выполнять поиск ресурсов с учетом регистра методами GetString и GetObject.
	ResourceSetType	Получает Type объекта ResourceSet объекта ResourceManager и использует для конструирования объекта ResourceSet.
