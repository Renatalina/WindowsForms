ContentAlignment - перечисление

Устанавливает выравнивание содержимого по поверхности рисунка.

Пространство имен:  System.Drawing

	BottomCenter -	Содержимое выравнивается вертикально по нижнему краю и горизонтально по центру.
	BottomLeft - 	Содержимое выравнивается вертикально по нижнему краю и горизонтально по левому краю.
	BottomRight -	Содержимое выравнивается вертикально по нижнему краю и горизонтально по правому краю.
	MiddleCenter -	Содержимое выравнивается вертикально по середине и горизонтально по центру.
	MiddleLeft -	Содержимое выравнивается вертикально по середине и горизонтально по левому краю.
	MiddleRight -	Содержимое выравнивается вертикально по середине и горизонтально по правому краю.
	TopCenter -	Содержимое выравнивается вертикально по верхнему краю и горизонтально по центру.
	TopLeft	-	Содержимое выравнивается вертикально по верхнему краю и горизонтально по левому краю.
	TopRight -	Содержимое выравнивается вертикально по верхнему краю и горизонтально по правому краю.
